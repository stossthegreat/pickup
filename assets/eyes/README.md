# Eye-contact lesson hero image

Drop **one** JPG here at exactly this path:

```
assets/eyes/lesson_eyes.jpg
```

Used by every Eye Contact (Presence) lesson as the gaze target the
user holds onto. Replaces the painted eyes / red dots.

## Format

- JPEG, **16:6 horizontal letterbox** (e.g. 1600 × 600 px), sRGB,
  ~85% quality, ≤ 500 KB.
- TIGHT crop on the **eyes only** — cheekbone to brow line.
  **No nose, no mouth, no hair.**
- Eyes **dead-centre**, **staring directly into the camera lens**.
  That's the entire intensity.
- Photoreal. No illustration.
- Dark / near-black background, single warm-red rim light on one
  side. Cinematic, *Drive* / *Dune* close-up energy.
- Lashes visible. Iris with depth (the limbal ring matters).
- Slight pupil dilation — sells the eye as "engaged."

Until you drop the file in, the lesson falls back to a black band
with a single red gleam at the centre so the build never breaks.
