AURALAY — MediaPipe FaceLandmarker model
=========================================

This directory ships the MediaPipe Tasks face_landmarker.task model that
backs the elite iris-level gaze detector.

TO ENABLE MEDIAPIPE GAZE (required once, not in version control to keep repo small):

    1. Download the model (~3.8 MB):

       curl -L -o face_landmarker.task \
         https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/latest/face_landmarker.task

    2. Place it in this directory (assets/models/face_landmarker.task).

    3. Run:  flutter pub get

    4. iOS only:  cd ios && pod install

    5. In the app, boot FaceDetectorService with:
         FaceDetectorService(engine: GazeEngine.mediapipe)

The model is REQUIRED for GazeEngine.mediapipe. Without it:
  - Android: plugin init() returns false → Dart throws, app should fall back to MLKit.
  - iOS:     plugin init() throws MODEL_MISSING → same fallback.
